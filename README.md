## schedule-fom

定时任务管理，相对于Spring Schedule提供了更多的功能支持

- 依赖

```xml
<dependency>
    <groupId>com.cowave.commons</groupId>
    <artifactId>schedule-fom</artifactId>
    <version>2.7.3</version>
</dependency>
```
